External albums

With Mediashare you can link-in albums from various external sources. You simply copy the album URL from your favorite photo application and insert it into Mediashare's album settings. After this you will see all the external photos as if they were uploaded to Mediashare.

Flickr

You can use the following URLs from Flickr:

  • Set: http://flickr.com/photos/USERNAME/sets/SET-ID/
  • Tags: http://flickr.com/photos/USERNAME/tags/TAG/
  • Tags: http://flickr.com/photos/tags/TAG/
  • Photo: http://www.flickr.com/photos/USERNAME/PHOTO-ID/in/set-SET-ID/
  • User: http://www.flickr.com/photos/USERNAME/

To use this feature, you must get an API key from Flickr. This key can be written into Mediashare's admin settings.

Smugmug

You can use the following URLs from Smugmug:

  • User: http://USERNAME.smugmug.com/
  • Gallery: http://USERNAME.smugmug.com/gallery/GALLERY-ID
  • Popular: http://USERNAME.smugmug.com/popular/#SOME-ID
  • Keyword: http://USERNAME.smugmug.com/keyword/KEYWORD#SOME-ID

To use this feature, you must get an API key from SmugMug. This key can be written into Mediashare's admin settings.

Picasa

You can use the following URLs from Picasa:

  • User: http://picasaweb.google.com/USERNAME
  • Album: http://picasaweb.google.com/USERNAME/ALBUMNAME
